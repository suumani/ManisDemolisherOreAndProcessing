# Mani's Demolisher Ore And Processing

Mani's Demolisher Ore And Processing は、**デモリッシャー撃破を起点**として、
「報酬・生産・UPS改善」を提供するエンドコンテンツ Mod です。

武器追加 Mod ではなく、**生産構造・物流・圧縮**によって工場を洗練させることを目的とします。

## この Mod がやること
- デモリッシャーから入手できる「デモリン鉱」系列の追加
- 精製・増殖・循環（ウラン遠心分離型の増殖と廃水回収ルート）
- 製錬 → 梱包材化 → サイエンスパック圧縮（輸送・保管・UPS改善）
- ボスサイエンスパックの生成（卵ルート／木ルート）

## 依存関係
この Mod は単体利用できません。以下が前提です：
- Manis_lib（共通処理）
- Manis_definitions（デモリッシャー定義）
- Mani's Boss Demolisher（ボス出現トリガ）
