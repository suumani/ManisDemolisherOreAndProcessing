# Mani's Demolisher Ore And Processing

Mani's Demolisher Ore And Processing は、**デモリッシャー撃破を起点**として  
「報酬・生産・UPS改善」を提供するエンドコンテンツ Mod です。

主な報酬は、**サイエンスパックの圧縮による輸送量削減**と、  
それを支える **デモリン鉱の生産・循環ライン**です。

本 Mod は武器を追加するものではなく、  
**生産構造・物流・圧縮**によって工場を洗練させることを目的とします。

---

## この Mod がやること
- デモリッシャーから入手できる「デモリン鉱」系列の追加
- 精製・増殖・循環  
  （ウラン遠心分離型の増殖と廃水回収ルート）
- 製錬 → 梱包材化 → サイエンスパック圧縮  
  （輸送・保管・UPS改善）
- ボスサイエンスパックの生成  
  （卵ルート／木ルート）

---

## 依存関係
この Mod は、以下の Mod を前提として設計されています：

- **必須**：Manis_lib（共通処理）
- **必須**：Manis_definitions（デモリッシャー定義）
- **強く推奨**：Mani's Boss Demolisher（ボス出現トリガ）

※ Mani's Boss Demolisher が未導入の場合でも動作しますが、  
　高速／巨大／キングといった上位デモリン鉱を利用する要素は利用できません。
